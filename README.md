# lab4

Flutter Contacts App (Firebase Auth + Firestore CRUD).

## Getting Started

### Prerequisites (Windows)
When building with Firebase/Google plugins on Windows, Flutter may require symlink support. Enable **Developer Mode**:

```powershell
start ms-settings:developers
```

Then toggle **Developer Mode** to ON and re-run `flutter run`.

### Firebase Platforms
This repo currently includes Firebase configuration for **Android** only (`lib/firebase_options.dart`).
